# PV-Quality-Complaince-dashboard
This Power BI dashboard analyzes pharmacovigilance case quality and compliance performance across safety case processing activities.

Business Problem

Pharmacovigilance teams must maintain strict quality standards while processing large volumes of safety cases.
Quality review teams need visibility into error types, associate performance, and compliance levels to ensure regulatory expectations are met.

This dashboard provides a structured view of quality metrics to identify performance gaps and support continuous quality improvement.

Key Metrics

• Total Cases Reviewed
• Critical Error Count
• Non-Critical Error Count
• Associate Compliance Percentage
• Quality Trend Analysis
• Error Distribution by Category

Data Source

The dataset simulates pharmacovigilance quality review data including case IDs, error classifications, review outcomes, and associate performance metrics used in drug safety quality monitoring.
